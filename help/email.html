Get help via email

We run three main mailing lists:

  • The announcements mailing list for announcing new releases and other important Wicket news.
  • The users mailing list for developers working with Wicket&emdash;that would be if you’re using Wicket or are interested in using it
  • The dev mailing list for talking about development of Wicket itself. If you doubt that your message belongs on this list, use the users list.
  • There’s also a commits mailing list for following SVN commits and JIRA issues.
